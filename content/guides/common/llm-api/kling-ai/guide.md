## 개요

Kling AI API를 사용하기 위한 키 발급 방법을 안내합니다.

## 1단계: Kling AI 플랫폼 접속

[Kling AI](https://klingai.com)에 접속하여 계정을 생성합니다.

## 2단계: API 키 발급

1. 대시보드에서 **API** 섹션으로 이동합니다.
2. **Create API Key**를 클릭합니다.
3. 키를 안전한 곳에 복사합니다.

## 3단계: 환경변수 설정

```env
KLING_API_KEY=발급받은_API_키
```
