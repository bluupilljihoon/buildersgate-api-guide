## 들어가기 전에

본 문서는 카드사 심사를 위한 **"결제경로 파일"**을 제작하기 위한 방법을 알려드리기 위해 작성되었어요.

- **결제경로 파일이란?**
  카드사심사를 위해, 상품의 **결제가 이루어지는 과정을 시각화하는 일**이에요.
- 결제경로 제작은 기본적으로 **"화면 캡처"**를 통해 제작 되어야 해요.
  본 가이드에 따라 누락되는 정보 없이 제작되어야 카드사 심사가 정상적으로 이루어 질 수 있어요.

## 결제경로 순서 및 도식화

결제경로는 아래 순서대로 캡처하고, 작업하면 됩니다.

| 단계 | 항목 | 설명 |
|:---:|------|------|
| ① | 가맹점 정보 기재 | 결제경로 제작 표지에 가맹점 정보를 기재해요. <br>> 상호명 / 사업자번호 / URL / 테스트계정 기재 |
| ② | 하단정보 캡처 | 필수정보가 모두 포함된 하단정보를 캡처해요. <br>> 상호명 / 사업자번호 / 대표자명 / 사업자주소 |
| ③ | 환불규정 캡처 | 환불규정을 캡처해요. |
| ④ | 로그인 / 회원가입 캡처 | 회원가입 경로 / 로그인 경로를 캡처해요. <br>※ 비회원 구매가 가능한 경우엔 생략해도 좋습니다. |
| ⑤ | 상품선택 / 구매과정 캡처 | 상품 구매하는 일련의 과정을 캡처해요. (유형, 무형 상품 모두 동일) |
| ⑥ | 카드 결제경로 캡처 | 카드 결제 과정을 캡처해요. |

## 결제경로 제작 유의사항

홈페이지 결제경로 제작 시, 아래 사항들을 꼭 지켜 주시길 바랍니다.

1. 반드시 **PPT 형식**으로 제작해주세요.
   추후 각 카드사 심사에 맞게 양식 수정이 필요하기 때문이에요.

2. 모든 캡처 화면은 북마크바 없이 **도메인이 보이도록 캡처**해주세요.
   도메인 주소가 상단에 노출 되어야 해요.

3. 모든 홈페이지 캡처 화면 시, PC의 **시간도 함께 캡처**가 되어야 해요.
   카드사 측에서 시간의 흐름을 확인하며 조작 여부를 판단하기 때문이에요.

4. 상품/서비스의 명칭, 이미지, 상세설명, 금액, 결제페이지, 결제창의 **흐름이 모두 확인**되어야 해요.

5. 상품 구매 시 사용하는 **테스트용 계정**과 **로그인 과정 캡처**가 필요해요.

6. **테스트용 결제창**으로 연동되어도 심사 가능해요.
   테스트용 연동 가이드: https://docs.tosspayments.com/guides/payment-widget/integration

7. **안심키인**을 사용하여 결제창이 없을 경우, **생략**해도 괜찮아요.

8. **비씨 카드**사의 경우, 결제 직전까지의 확인이 필요하기에 각 카드사 인증 화면까지 캡처해야 해요.

## ① 가맹점 정보 기재

시작 페이지에 가맹점 정보를 기재해요.

**구성 항목:** (1) 상호명 / (2) 사업자번호 / (3) 가맹점 URL / (4) 테스트 ID/PW (없을 시 생략)

**예시**

| 항목 | 내용 |
|------|------|
| (1) 상호명 | 토스페이먼츠 |
| (2) 사업자번호 | 123-45-678 |
| (3) URL | https://www.tosspayments.co |
| (4) Test ID | test.naver.com |
| (5) Test PW | 123456789 |

## ② 하단 정보 캡처

**필수 구성 항목:** (1) 상호명 / (2) 대표자명 / (3) 사업자등록번호 / (4) 통신판매업신고번호 / (5) 사업장주소 / (6) 유선전화번호

## ③ 환불규정 캡처

### 유형상품 예시

**교환 및 반품 정보**

- 환불 및 교환은 수령하신 날을 포함하여 7일 내에 요청해주셔야만 가능합니다. (홈페이지 내 Q&A 접수)
- 오염 및 미세한 잡사, 착용의 흔적, 수령 후 7일 이후 환불/교환 신청은 불가하니 참고부탁드립니다.
- 단순변심은 왕복 배송비 5,000원이 청구됩니다.
- 색상 교환 및 고객 변심은 고객 부담 왕복 배송비입니다. (사이즈 교환 포함)

**배송정보**

- 배송방법: 택배
- 배송지역: 전국 지역

### 무형상품 예시

**제 19조 결제 취소 & 환불**

① 회사와 구매에 관한 계약을 체결한 회원은 아래와 같이 결제에 대한 취소 및 환불을 요구할 수 있습니다.
- 이용계약의 신청 후, 회사로부터의 상담이 제공되지 않은 경우, 결제취소가 가능합니다.
  - **a. 정기결제회원:** 환불은 서비스를 이용한 일수를 제외하고 일할 계산으로 진행됩니다. 월 기준은 30일이고 월 이용료를 30으로 나눈 금액을 말합니다. 결제시간으로부터 24시간 이후 ~ 15일까지는 남은 일 수에 대한 일할 계산으로 처리됩니다.
  - **b. 연간결제회원:** 연 기준은 12개월이고, 잔여이용료는 전체 연간결제 이용료를 12로 나눈 하루 일에 대한 금액이 환불됩니다.
- 회원이 상담 후 상대 프로필카드를 2회 이상 수령한 경우, 잔여횟수가 남은 회원에 한하여 이용 금액과 위약금 10%를 제외한 부분 환불이 가능합니다.

② 회사의 귀책사유로 결제 오류가 발생한 경우
③ 회사의 귀책사유로 서비스가 중단되는 경우

> 본 조의 환불 금액 기준은 연간결제 회원이라 하더라도 정기결제 금액으로 계산 후 진행됩니다.

## ④ 로그인 / 회원가입 캡처

로그인 혹은 회원가입 경로를 캡처해요.
비회원 구매가 가능할 경우, **"비회원 구매 경로"**를 캡처해요.

**로그인 화면에 포함되어야 할 요소**
- 아이디 / 비밀번호 입력 필드
- 로그인 유지 체크박스
- 로그인 버튼
- 회원가입 링크

**비회원 구매 화면 (해당 시)**
- "비회원 구매" 버튼
- 비회원 구매 안내 문구
- 회원가입 유도 문구

## ⑤ 상품 선택 / 구매과정 캡처

유/무형 상품의 구매 경로를 모두 캡처해주세요.

**예시 경로:** shop > 카테고리 선택 > 제품 선택 > 옵션 선택 > 구매하기 탭 등

### 상품 목록 페이지 캡처

상품명, 가격이 노출된 상품 목록 화면을 캡처해요.

### 장바구니 페이지 캡처

| 항목 | 내용 |
|------|------|
| 상품명 | 토스페이먼츠 북 |
| 옵션 | 에디션 / 일반 1개 |
| 상품가액 | 22,900원 |
| 배송비 | 3,000원 |
| **총 주문가액** | **25,900원** |

### 주문 / 결제 페이지 캡처

- **주문 상품 정보:** 상품명, 옵션, 금액 확인
- **주문자 정보:** 이름, 연락처, 이메일
- **배송 정보:** 수령인, 연락처, 주소, 우편번호
- **주문 요약:** 상품가격 + 배송비 = 총 주문금액
- **결제수단:** 신용카드 / 무통장입금 등
- **"결제하기" 버튼** 노출 확인

## ⑥ 카드 결제경로 캡처

### 기본 카드 결제창 캡처

카드 결제경로를 캡처해주세요.

**결제창에 표시되는 카드사 예시**
토스페이 / 페이북 / 현대 / 신한 / 바비(대이북) / KB국민(KB pay) / 삼성 / 롯데 / 하나 / NH채움 / 우리 / 수협

**동의 항목 확인**
- 전체 동의하기
- 전자금융거래 기본약관
- 개인정보 수집 및 이용 동의
- 개인정보 제3자 제공 연내

### a. 비씨 카드의 경우 (추가 캡처 필요)

> 비씨 카드의 경우 실제 카드결제 화면 창이 필요해요!

아래처럼 비씨 카드 결제경로를 캡처해주세요.

- 페이북 결제창: 비밀번호 결제 / 기타 결제 탭
- "기타 결제" 탭 선택 후 화면까지 캡처

### b. 빌링결제의 경우 (추가 캡처 필요)

> 빌링결제의 경우 정기결제용 카드 입력창까지 캡처해야 해요.

**카드 정보 입력창**
- 카드번호
- 유효기간 (MM/YY)

**본인 인증 입력창**
- 이름
- 주민등록번호 앞 7자리
- 통신사 선택
- 휴대폰번호
- 인증번호 받기
