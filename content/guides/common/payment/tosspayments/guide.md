## 개요

토스페이먼츠 결제 연동을 위한 가맹점 신청부터 API 키 발급까지의 과정을 안내합니다.

## 1단계: 토스페이먼츠 가입

[토스페이먼츠](https://www.tosspayments.com)에 접속하여 회원가입합니다.

## 2단계: 가맹점 신청

1. **가맹점 신청**을 클릭합니다.
2. 사업자 정보를 입력합니다.
3. 필요 서류를 제출합니다.

## 3단계: 테스트 키 확인

1. 가입 즉시 **테스트** 모드에서 사용 가능한 키가 제공됩니다.
2. **개발자센터** > **API 키**에서 확인합니다.

## 4단계: 라이브 키 발급

1. 가맹점 심사가 완료되면 **라이브** 키가 활성화됩니다.
2. 클라이언트 키와 시크릿 키를 복사합니다.

## 5단계: 환경변수 설정

```env
TOSS_CLIENT_KEY=test_ck_테스트_클라이언트_키
TOSS_SECRET_KEY=test_sk_테스트_시크릿_키
```
