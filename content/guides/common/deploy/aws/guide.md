## 개요

AWS 계정을 생성하고 카드를 연결하는 방법을 안내합니다.

## 1단계: AWS 가입

[AWS](https://aws.amazon.com)에 접속하여 **Create an AWS Account**를 클릭합니다.

## 2단계: 계정 정보 입력

1. 이메일, 비밀번호, 계정 이름을 입력합니다.
2. 연락처 정보를 입력합니다.

## 3단계: 결제 수단 등록

1. 신용카드 또는 체크카드 정보를 입력합니다.
2. $1 임시 결제로 카드 유효성이 확인됩니다.

## 4단계: 본인 확인

1. 전화번호를 입력하고 인증 코드를 받습니다.
2. 코드를 입력하여 본인 확인을 완료합니다.
