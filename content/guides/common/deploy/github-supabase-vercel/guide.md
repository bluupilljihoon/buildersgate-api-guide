## 개요

GitHub, Supabase, Vercel 계정을 생성하고 카드를 연결하는 방법을 안내합니다.

## GitHub 계정 생성

1. [GitHub](https://github.com)에 접속합니다.
2. **Sign up**을 클릭하고 계정을 생성합니다.
3. 이메일 인증을 완료합니다.

## Supabase 계정 생성

1. [Supabase](https://supabase.com)에 접속합니다.
2. **Start your project**를 클릭합니다.
3. GitHub 계정으로 로그인합니다.

### Supabase 결제 수단 등록

1. **Organization Settings** > **Billing**으로 이동합니다.
2. **Add Payment Method**에서 카드 정보를 입력합니다.

## Vercel 계정 생성

1. [Vercel](https://vercel.com)에 접속합니다.
2. **Sign Up**을 클릭하고 GitHub 계정으로 가입합니다.

### Vercel 결제 수단 등록

1. **Settings** > **Billing**으로 이동합니다.
2. **Add Payment Method**에서 카드 정보를 입력합니다.
